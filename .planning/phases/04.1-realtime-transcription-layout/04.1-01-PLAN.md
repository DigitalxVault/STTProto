<task type="auto">
  <name>Task 2: CSS for fixed bottom layout with column-reverse transcript</name>
  <files>style.css</files>
  <action>
Update the CSS to implement the bottom-anchored layout:

1. **Remove** the old `.transcript-panel` flex:1 rule (it currently fills the middle of the screen).

2. **Add `.bottom-bar`** — fixed to the bottom of the viewport:
```css
.bottom-bar {
  position: fixed;
  bottom: 0;
  left: 0;
  right: 0;
  display: flex;
  align-items: flex-end;
  padding: 0 1rem;
  padding-bottom: env(safe-area-inset-bottom, 16px);
  gap: 0.75rem;
  z-index: 10;
  pointer-events: none;
}
.bottom-bar > * {
  pointer-events: auto;
}
```

3. **Update `.transcript-panel`** to be the left side, growing upward:
```css
.transcript-panel {
  flex: 1;
  display: flex;
  flex-direction: column-reverse;
  max-height: 50dvh;
  overflow-y: auto;
  padding: 0.75rem;
  background: transparent;
  border: none;
  font-size: 13px;
  -webkit-overflow-scrolling: touch;
}
```
Note: Use `column-reverse` so `appendChild` naturally places new items at the visual bottom (which appears at the bottom of the reversed column = nearest to the bottom bar). Remove the old background/border/margin since the transcript should float transparently over the dark background.

4. **Update `.mic-container`** to sit at flex-end (right side) of the bottom bar:
```css
.mic-container {
  flex-shrink: 0;
  display: flex;
  align-items: flex-end;
  padding: 0 0 0.5rem;
}
```

5. **Update `.transcript-entry`** — keep existing styling but ensure entries are readable against dark background (they already use --text-primary which is light).

6. **Update `.transcript-placeholder`** — no changes needed, just ensure it still works with column-reverse.

7. **Remove** `padding-bottom: env(safe-area-inset-bottom, 16px)` from `body` since the bottom-bar now handles safe area. Keep `padding-top: env(safe-area-inset-top, 0px)` on body.

8. **Remove** the `.state-demo` section CSS (Phase 1 demo strip) — it was only for Phase 1 verification and is no longer in the HTML.

9. Add a subtle text shadow on transcript entries for readability against the dark background:
```css
.transcript-entry {
  margin-bottom: 0.4rem;
  color: var(--text-primary);
  word-wrap: break-word;
  text-shadow: 0 1px 3px rgba(0,0,0,0.8);
}
```
